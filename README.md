# Приложение для того, чтобы ботать ЕГЭ русский

пока тут только ударения, потом посмотрим


## Установка

Необходимо установить все зависимости:

```commandline
pip install -r requirements.txt
```

(там пока только `Flask`)


## Запуск

Просто запустить файл `app.py`, можно через командную строку, можно просто так

Сервер по умолчанию запускается на `127.0.0.1:5000`
